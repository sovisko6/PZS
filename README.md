Počítačové zpracování signálů (PZS)
Tento repozitář obsahuje řešení semestrálních projektů zaměřených na digitální analýzu biosignálů a řeči.

Obsah repozitáře
Repozitář je rozdělen do dvou hlavních částí podle zadání seminárních prací:

1. Detekce tepové frekvence z EKG
Adresář: Kód pro první seminární prác

Popis: Algoritmus pro zpracování elektrokardiogramu (EKG). Program načítá surová data, provádí filtraci šumu a detekuje komplexy QRS (zejména R-špičky).

Výstup: Vizualizace signálu s označenými špičkami a výpočet průměrné tepové frekvence (BPM).

2. Analýza patologické řeči
Adresář: Kód pro druhou seminární práci

Popis: Nástroj pro analýzu akustických parametrů řečového signálu. Zaměřuje se na identifikaci znaků typických pro hlasové poruchy nebo patologie (např. Parkinsonova choroba).

Analyzované parametry:

Jitter & Shimmer: Variabilita frekvence a amplitudy.

HNR (Harmonics-to-Noise Ratio): Poměr harmonických složek k šumu.

Spektrální analýza: Sledování formantů a energetického spektra.
